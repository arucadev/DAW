Operació i assignació (+=, -=, *=, /=):

- a += b **→** a = a + b
- a -= b **→** a = a - b
- a *= b **→** a = a * b
- a /= b **→** a = a / b

Aquests operador realitzen la operació entre la variable de l'esquerra i una constant, variable o literal, i guarden el resultat en la variable de l'esquerra:

	let a = 2;  
	let b = 3;  
	a += b;        // Equival a **a = a + b;** → a = 5
