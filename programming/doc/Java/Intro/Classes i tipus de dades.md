Tipos de nums enteros:
- byte => 0 - 255
- short => -32000 - +32000
- int => -2000M - +2000M
- long => 2^64

Tipos de nums decimales:
- float => decimales generales
- double => decimales precisos

Tipos de strings:
- char 
  `char inicial = 'A';  // Va con comillas simples`
- String
  `String nombre = "Alex"`
